## Installation

EasySwiftUIKit is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```podInstallEasySwiftUIKit01
pod 'EasySwiftUIKit'
```
or
```podInstallEasySwiftUIKit02
pod 'EasySwiftUIKit', '~> 1.0'
```
